
# TaskTrek 🚀

**TaskTrek** is a simple, clean task tracker built with React and localStorage. It lets you add and delete tasks, and it remembers your to-dos even after refreshing the page.

## 🔧 Tech Stack

- React 
- HTML + CSS
- localStorage API
- GitHub Pages for deployment
